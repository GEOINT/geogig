
.. _geogig-geopkg-list:

geogig-geopkg-list documentation
################################



SYNOPSIS
********
geogig geopkg list [options]


DESCRIPTION
***********

This command simply lists of the tables available in the specified GeoPackage database.

OPTIONS
*******    

-D, --database  GeoPackage database to use.  Default: database.gpkg

-U, --user      User name to use when accessing the database.  Default: user
